Getting Started

Big Daddy V3 is a powerful WhatsApp bot that allows you to automate messages, manage groups, and much more.

To get started, you'll need to:

  1. Select your preferred command prefix
  2. Enter your WhatsApp number with country code
  3. Use the pairing code to connect your number
  4. Wait for automatic connection verification

Deploying Your Bot

To deploy your bot:

  1. Click on "Deploy Bot" in the sidebar
  2. Select your preferred command prefix (., /, !, or ')
  3. Enter your WhatsApp number with country code (without spaces)
  4. Submit and wait for the pairing code
  5. Use the pairing code in your WhatsApp
  6. System will automatically verify connection status

Your bot will be ready to use once connection is verified.

Available Commands

Once your bot is deployed, you can use various commands with your selected prefix:

[prefix]help - Show all available commands
[prefix]menu - Display the main menu
[prefix]sticker - Create stickers from images
[prefix]ai - Chat with AI
[prefix]play [song] - Play music
[prefix]tts [text] - Convert text to speech

Replace [prefix] with your selected prefix (., /, !, or ')

Automatic Connection Monitoring

The system automatically monitors your bot connection:

  • Real-time verification: Backend checks connection every 1 minute (3 attempts)
  • Automatic status updates: Frontend updates in real-time via server-sent events
  • Hourly monitoring: All connected numbers are checked every hour
  • Bulk verification: System-wide checks every 6 hours
  • Works when closed: Verification continues even if you close the web app

Troubleshooting

Bot not responding?

  • Check if your bot is still running in the dashboard
  • Verify your internet connection
  • Make sure you're using the correct prefix
  • Try restarting the bot from the console

Can't pair your number?

  • Ensure you entered the correct number with country code
  • Check that the pairing code hasn't expired
  • Try restarting the deployment process
